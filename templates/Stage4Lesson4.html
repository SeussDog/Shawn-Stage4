4.4 Python Break-Modulus and Dictionaries

Modulus Operator = %

  • Format = <number> % <modulus> gives us the <remainder>

String vs. List vs. Dictonary

String
  • 'hello'
  • this is a sequence of chracters
  • it is immutable
    • s[i]th character is 's'
List
  • [‘alpha’, 23]
  • this is a list of elements
  • it is mutable
    • p[i]th
      • ith element of p
    • p[i]th = u
      • replace value of ith element with u.
Dictionary
  • {‘hydrogen’: 1, ‘helium’: 2}
  • this is a set of <key, value> pairs
  • pairs are mutable
    • d[k]
      • k is the key value associated with k in d.
    • d[k] = v
      • update k with v.
