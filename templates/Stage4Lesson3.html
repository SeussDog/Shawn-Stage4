4.2 Forms

Definition

Forms are what users use to enter text, select from dropdown menus, check checkboxes, etc...

HTML Forms

  <form>
      <input name = “name”> <— example = <input name = “q”>
  </form>
The following gives us a 'submit' button. This is not different than just hitting <enter> on the keyboard 
  <input type = “submit”>
The following will open whatever URL we specify and pass our input to it. 
  <form action = “/foo”>
The following shows the text as dots to protect the data entry, though the values do get passed to the URL and can be viewed. 
  <input type=“password”>
The following shows a checkbox. The URL shows either "on" or it doesn't appear at all in the URL. 
  <input type="checkbox">
The following acts just like a checkbox except you can't uncheck them, unless you give each radio button the same name. Then you are only allowed to select one at any given time. 
  <input type="radio">
The following allows us to give the radio button a value that gets sent to the server. 
  <input type="radio" name=q value="one">
The following allows us to name the radio button. 
  <label>
The following allows us to create a dropdown box with these 3 values. When we add "value" to option...that value is what gets passed to the URL. 
  <select name=q>
      <option value=“1">one</option>
      <option>two</option>
      <option>three</option>
