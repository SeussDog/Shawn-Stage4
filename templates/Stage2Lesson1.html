Programming

Toaster vs. Computer
  • A toaster is made of mechanical parts that repeat the same tasks over and over.
  • A computer uses programs to find their function. The programs have repeatable and instructions sets to provide the computer function.

Python

It is a ligh level program or what we call an Interpretor. It takes code and translates it into another programming language. Python can be used for creating HTML code quickly.

Search Engines

These are used every day for searching the web. The 3 main components for building a search engine are:
  • Find Data
  • Build an Index
  • Rank Pages

Print

In programming this is a way of outputting data. In Python this is a command for outputting text or mathematical problems.

Natural Language vs. Programming Language

  • Natural Language is ambiguous. Words like "bi-weekly" may mean multiple things.
  • Natural Language is verbose.
  • Programming languages have precise grammar also known as “syntax”.
