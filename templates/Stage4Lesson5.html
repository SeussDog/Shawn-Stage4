4.5 Working with App Engine

Differences between Get vs. Post

Get
  • parameters in URL
  • used for fetching documents
  • maximum URL length
  • ok to cache
  • shouldn't change the server
Post
  • parameters in body
  • used for updating data
  • no maximum length
  • not ok to cache
  • ok to change the server
