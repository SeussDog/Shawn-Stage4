What is a Variable?

A value that can vary. It is convention to use lower case for variables, but not a rule. Variables are useful for summarizing a value and can be used over and over to simplify our code.

Assigning a value to a variable:

What's the difference between what the equals sign = means in 2+3=5 and my_variable=5

In programming the equals sign means "takes the value of".

What is the difference between 2 + 2 and "2" + "2"?

2 + 2 would print out as 4 in Python, whereas "2" + "2" would concatenate these two values as strings and print out 22.

Indexing Strings
This is essentially a way to output a letter from a string by indicating which letter to output from the string. For example:

<string> [<expression>]

'udacity' [0]

0123456

name = udacity

print name [1]

d

print name [-1]

Selecting Sub-Sequences
  • <string> [<expression>] --> one-character string, where the expression is a # like before.
  • <string> [< expression> : <expression>] --> expressions are numbers and this will result in a subsequence of the characters. The 1st expression is the 1st letter. The 2nd expression is the Final Letter in the combined string.
Method
A built in procedure used in Python, such as "Find".
How to Use Find
<string>.find(<string>) --> number that gives first position in search string where the target string appears. If the target string is not found, the output will be -1. For Example:

pythagoras = 'There is a geometry in the mumming of the strings, there is music in the spacing of the spheres.'

print pythagoras.find('string')

40 is the result

Parameter
  • Something that goes inside the parentheses. For example: "text".find("t",1)
  • By adding the comma and # parameter to the find Method we get the ability to search from a defined starting point in a string.
