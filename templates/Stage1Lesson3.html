Avoiding Repetition

CSS allows us to avoid repetition by applying a style to a class of text. This allows us to avoid adding redundant HTML formatting tags.

CSS Sumamry and Reference

There are 3 methods for including CSS Styling on a web page.
Method 1: Write CSS in the <head> of your HTML

* This is good for small projects

Method 2: Link your HTML to a separate CSS file

* This let's you stay more organized on larger projects.
