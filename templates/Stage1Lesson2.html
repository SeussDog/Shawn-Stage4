Stage 1: Lesson2: Creating a Structured Document

HTML

HTML controls the "structure" of a web page. It is the language of the Internet. Tags are the words. These tags form a tree like structure. The tags can have attributes with values.

CSS

Cascading Style Sheets (CSS) is used to control the "style" of a page (how it looks). It uses syntax and rules to define styles.

DOM

Document Object Model (DOM) is a tree like structure found in HTML.
