Mutation and Aliasing

Strings are immutable whereas Lists are mutable.
Mutation: You can't mutate a number, but you can mutate a list. Examples: 

  p = ['H','e','l','l','o']
  print p = Hello
  p[0] = 'Y'   ****This changes a value of the list****
  print p = Yello
  q = p
  q [4] = '!' ****This changed the value of q and p****
Aliasing Two variables can share the same value.

Index

<list>. index (<value>)

if <value> is in the <list>, returns the first position where <value> is found in <list>. Otherwise it produces an error.

Example:


            p = [0,1,2]
            print p.index(2)

This returns 2 and it only returns the first matching value. If the value is in the list, it will return the value where it is first found. The problem is if the value is not in the list, it returns an error.

Example 2:


            def find_element (p,t):
              if t in p:
                return p.index(t)
              else:
                return -1
            print find_element([1,2,3],3)
            #>>> 2
            print find_element(['alpha','beta'],'gamma')
