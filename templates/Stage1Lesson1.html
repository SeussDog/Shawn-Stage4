Stage 1: Lesson 1: Basics to Web and HTML

The Web

The web is a collection of HTML documents. Hyperlinks connect all the HTML documents together creating a web like structure. There are about 30 billion web pages.

HTML

This stands for Hyper Text Markup Language. HTML is made up of text, markup, references to other documens (for example pictures and videos) and links to other pages.

Markup

The following are examples of different types of markup:
  • Tags = composed of the name of the element, surrounded by angle brackets. An end tag also has a slash after the opening angle bracket, to distinguish it from the start tag.
  • Elements = The contents that sits between tags.
  • Attributes = are used inside some tags like the Anchor tag <a>. the attribute in this case is an href. Example = <a href = "www.reddit.com"> derp </a>
  • Images = image files. For example <img> or <img src = "url" alt = "txt"> (This is for broken link messages)
  • Line Breaks = these are carriage returns. For Example <br> (line break) or <p> (paragraph)
  • Inline vs Block = These are the two types of breaks. <br> is inline and <p> is block. Inlines are just lines of text whereas blocks, create a block of extra pixel space around the line.
