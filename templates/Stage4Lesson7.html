4.7 HTML Templates

Definition

A template library is a library to build complicated strings (html). The following are resources for building HTML templates.
  • jinja2 (built into Google App Engine)
  • jinja.pocoo.org

Variable Substitution

The following means print in Jinja:
{% raw %} 
{{
{% endraw %}
The following is a Jinja variable:
{% raw %} 
{{ variable }}
{% endraw %}

Statement Syntax

{% raw %} 
{% statement %} 
  output
{% end statement %}
{% endraw %}
The following is an example:
{% raw %} 
{% if name == “Shawn” %}
  Hello, Shawn!
{% else %}
  who are you?
{% end if %}
{% endraw %}

For Loop Syntax

{% raw %} 
{% for statement %} 
body
{% endfor %}
{% endraw %}

Helpful Tips

  • Always automatically escape variables when possible
  • Minimize code in templates
  • Minimize code in coe

Template Inheritance

You don't want to write the same header and footer for each page.

For Example:

base.html (the folowing inherit this template)

  • shopping-list.html
  • fizzbuzz.html

Benefits of Templates

  • seperate different types of code
  • make more readable code
  • more secure websites
  • html that is easier to modify
